1. Vec::new()
This creates an empty vector.
let mut v: Vec<i32> = Vec::new();
v.push(1);
v.push(2);
v.push(3);
You have to add elements manually later with .push().

2. vec![1, 2, 3]
This is the macro version (note: lowercase vec!, not Vec!).
It creates a vector already filled with those values.
let v = vec![1, 2, 3];
It’s just shorthand for the first method.

So:
Vec::new()	Create an empty vector	✅	When you’ll push later
vec![1, 2, 3]	Create a vector with initial values	❌	When you know values upfront
